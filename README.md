# V1-Drift
Python code to replicate drift in human's V1 article:

Roth, Z.N., Merriam, E.P. Representations in human primary visual cortex drift over time. Nat Commun 14, 4422 (2023).
 https://doi.org/10.1038/s41467-023-40144-w

using python package for multi-scale image processing, adapted from Eero Simoncelli’s matlabPyrTools: https://pyrtools.readthedocs.io/en/latest/
